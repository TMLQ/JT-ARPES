样品台和高精度掩模操作
=========================
样品台和高精度XY掩模板控制集成在一个软件内，使用时需确保两者控制器都打开。

本设备配有两个垂直方向的高精度淹没板，通过软件控制，界面和样品台集成在一起。掩模版运动通过软件上的1和2控制，可实现普通的掩模生长功能。如果需要进行梯度生长，请联系管理员，一起观察，更改掩模版的运动速度。
